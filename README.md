# Mahammad Sheykhov — CV & Portfolio

**Live Site**: https://deeeeniz.github.io/site1101/

## Overview
Multi-page Jekyll portfolio showcasing academic projects, technical skills, and professional experience. Features responsive design, interactive project carousel, and comprehensive social media integration.

## Local Development
```powershell
# Build the site
jekyll build

# Serve locally (for development use empty baseurl)
jekyll serve --baseurl ""
```

## Key Features
- **SITE 1101 Team Project**: Interactive 6-image carousel showcasing digital logic gates implementation
- **Responsive Design**: Mobile-first approach with breakpoints at 1024px, 768px, and 480px  
- **Social Integration**: GitHub, Codecademy, Codeforces, LeetCode, LinkedIn
- **Multi-page Navigation**: Home, About, Education, Projects, Experience, Skills, Languages

## Deployment
Site automatically deploys to GitHub Pages when changes are pushed to `main` branch.

## Files Structure
- `index.md`, `about.md`, etc. — Page content
- `_layouts/default.html` — Page template
- `ADA_CSS.css` — Site styles
- `ADA_ProjJS.js` — Interactive functionality
- `_config.yml` — Jekyll configuration
